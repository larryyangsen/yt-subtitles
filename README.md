# yt-subtitles

## Watch two different subtitles in the same time , if exist two or many subtitles

## How to run

-   npm install
-   npm run build
-   npm start
-   open http://localhost:8080

    -   youtube-dl

-   frontend

    -   axios
-   builder
    -   parcel-bundler
